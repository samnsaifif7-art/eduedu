## ✨ Features

### 💬 EduBot RAG Q&A Engine
- Upload any NCERT or educational PDF
- Semantic search using **FAISS** vector index + **Sentence Transformers**
- Answers generated by **Llama-3 via Groq API** (free, ~500 tokens/sec)
- Extractive fallback if no API key — always works offline
- Confidence scores, source citations, response time tracking

### 📊 Student Performance Predictor
- **GradientBoosting + RandomForest Ensemble** — 88-90% accuracy
- Trained on 8,000 synthetic student records with realistic correlations
- Predicts grade (F → A+), risk level, and expected score
- 4 engineered features: study×attendance, prev×assignments, consistency score
- Personalised improvement recommendations per student

### 🗺️ Adaptive Topic Recommender
- Content-based filtering using **TF-IDF cosine similarity**
- Covers NCERT curriculum for Grades 6–10 across 5 subjects
- Prioritises weak subjects, skips completed topics
- Generates complete **7-day personalised study plan**

### ⚡ Production Backend
- **FastAPI** REST API with auto-generated Swagger docs at `/docs`
- Request logging middleware with timing
- Global error handling with detailed responses
- Model pre-warming at startup
- **CORS enabled** for HTML frontend

## 🧠 ML Pipeline Details

### RAG Engine Flow
```
PDF Upload → PyPDF2 Parse → 250-word Overlapping Chunks
    → Sentence Transformers (all-MiniLM-L6-v2) Embeddings
    → FAISS IndexFlatIP (cosine similarity)
    → Top-K Retrieval → Groq Llama-3 Answer Generation
```

### Performance Predictor Flow
```
Student Data (7 features) → Feature Engineering (4 new features)
    → GradientBoosting (400 trees, lr=0.05) + RandomForest (300 trees)
    → Soft Voting Ensemble → Grade + Risk + Feature Importance
```

### Topic Recommender Flow
```
Student Profile → Weak Subject Priority Boost
    → TF-IDF Cosine Similarity on Topic Descriptions
    → Curriculum Filter (skip completed) → Ranked Topics + 7-Day Plan
```

## 🚀 Quick Start

### Prerequisites
- Python 3.11+
- Git
- Free Groq API key from [console.groq.com](https://console.groq.com)

### 1. Clone the repository
```bash
git clone https://github.com/YOUR_USERNAME/edumind-ai.git
cd edumind-ai
```

### 2. Create virtual environment
```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Mac/Linux
python -m venv venv
source venv/bin/activate
```

### 3. Install dependencies
```bash
# Windows — install PyTorch CPU build first
pip install torch --index-url https://download.pytorch.org/whl/cpu
pip install -r requirements.txt
```

### 4. Set up environment
Create `backend/.env` file:
```
GROQ_API_KEY=your_groq_api_key_here
```

### 5. Generate sample dataset
```bash
python data/generate_dataset.py
```

### 6. Start the backend
```bash
cd backend
uvicorn main:app --reload --port 8000
```

You should see:
```
✅ Sentence-transformers ready.
✅ Groq/Llama-3 ready.
Training improved model on 8000 samples...
Train: 94% | Test: 89%
INFO: Uvicorn running on http://127.0.0.1:8000
```

### 7. Open the frontend
Open `frontend/index.html` in your browser.

### 8. Access API docs
Visit **http://localhost:8000/docs** for interactive Swagger UI

## 📡 API Reference

### RAG Q&A

#### Upload PDF
```
POST /api/rag/ingest
Content-Type: multipart/form-data
file: <pdf_file>
```
```json
{ "status": "success", "filename": "ncert_science.pdf", "chunks_indexed": 142 }
```

#### Ask Question
```
POST /api/rag/ask
{ "question": "What is photosynthesis?", "top_k": 3 }
```
```json
{
  "answer": "Photosynthesis is the process by which...",
  "sources": ["ncert_science.pdf"],
  "confidence": 0.87,
  "model_used": "Llama-3 (Groq)",
  "response_time_ms": 1240.5
}
```

### Performance Prediction
```
POST /api/predict/performance
{
  "attendance_pct": 85, "study_hours_per_day": 3.5,
  "prev_exam_score": 72, "assignments_completed_pct": 80,
  "sleep_hours": 7, "extracurricular_activities": 2, "subject": "Math"
}
```
```json
{
  "predicted_grade": "B",
  "predicted_score": 76.4,
  "risk_level": "Low",
  "recommendations": ["Increase study time to 4+ hours for grade A."],
  "feature_importance": { "prev_exam_score": 0.28, "attendance_pct": 0.22 }
}
```

### Topic Recommendation
```
POST /api/recommend/topics
{
  "student_id": "STU001", "completed_topics": ["Integers"],
  "weak_subjects": ["Math"], "grade_level": 9, "learning_style": "visual"
}
```

## 🔧 Common Issues & Fixes

| Error | Fix |
|-------|-----|
| `fbgemm.dll not found` | `pip install torch --index-url https://download.pytorch.org/whl/cpu` |
| `Cannot import module main` | `cd backend` before running uvicorn |
| `Groq proxies error` | `pip install groq==0.11.0` |
| `No GROQ_API_KEY` | Create `backend/.env` with your key |
| `Port 8000 in use` | `uvicorn main:app --port 8001` |
